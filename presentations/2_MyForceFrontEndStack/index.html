Application priorities:

  • UX
  • Maintainable by multiple developers
  • Extendable
  • VanillaJS
  • jQuery

are no options

The obvious trend is to have SPA's as these can give the most native feeling experience.

Another trend is to have an MVC framework as this allows us to consume REST api's.

Only 2.5 real options:

  • React
  • Angular 1.x
  • Angular 2

Angular 2 is considered a new framework. They've learned a lot from React and syntax needed to change

We ignore Angular2 as at the moment this is only a .5 option

pro angular:

  • easier to find developers
  • better out of the box experience

pro react:

  • Better performance
  • More configurable

Layout frameworks:

  • Bootstrap
  • Material
  • Foundation

We use bootstrap but try to lose the bootstrap feel

ES version:

  • Browser support?
  • Transpiling to previous version?
  • PolyFilling support for next version

We used supported version by browser (ES5)

Now we use ES6 transpiled by Babel to ES5 polyfilled for IE9/10

We need a generator that offers:

  • Angular 1
  • ES6 or later with transpiling
  • Bootstrap

Some other things we want:

  • Bower support
  • Gulp support
  • Sass support
  • Linting support
